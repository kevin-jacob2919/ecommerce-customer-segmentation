# E-commerce Customer Segmentation

## Project Overview

This project performs customer segmentation for an e-commerce business using machine learning clustering techniques.

## Objective

Group customers based on purchasing behavior so businesses can design targeted marketing strategies.

## Steps Performed

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Selection
4. Customer Segmentation using Clustering

## Algorithms Used

* K-Means Clustering

## Tools & Libraries

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Outcome

Customers were segmented into groups based on spending patterns and purchasing behavior, enabling more effective marketing strategies.
